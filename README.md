# GraphQL Server Boilerplate

## Try

### Install packages

```bash
yarn
```

### Set env!

Change .env.example file then rename it to .env

### Go dev!

```bash
npm run dev
```

### Go prod!

```bash
npm run build
npm run start
```

### Tips

The `package.json` file will be your best friend.
